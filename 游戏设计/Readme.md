####设计思路

一个游戏的主大厅，为新手村庄，在这里玩家可以进行一系列的准备，列如锻造武器等等，也提供一定的养成休闲玩法，例如挖矿，种菜，货币交易等等，村庄剧情由此处出发，引入相关背景，如果时间允许可以添加更多的细节
